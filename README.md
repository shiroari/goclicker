# Introduction

Go-clicker is simple crawler tool to find broken pages. 

# Run or Build

Make sure you have current project in you `GOPATH`

Run without build:

`go run robot/main.go`

Or you can build it and make binary file:

`go build -o goclicker robot`

`./goclicker`
